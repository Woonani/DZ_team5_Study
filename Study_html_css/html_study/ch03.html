시맨틱 태그

일반적인 웹사이트 문서 구조는 header(헤더) - main(본문) - footer(푸터)영역으로 나뉩니다.

nav태그는 다른 웹 문서로 연결하는 링크를 만듭니다.

nav태그는 위치가 자유로워서 헤더나 푸터나 사이드바 안에 포함할 수도 있고 독립해서 사용할수도 있습니다.

nav태그 여러개 사용할 경우 각각 id 속성을 지정하면 내비게이션마다 다른 스타일 적용할 수 있습니다.
